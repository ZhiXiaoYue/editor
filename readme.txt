本程序可实现打开旧文件与新建画作的功能
1、打开文件
   按照提示输入文件路径，按回车确认后再输入文件名加
   扩展名，按回车键确认。如果无该文件，将会提示您“
   无该文件”并退出。
2、新建画作
   该项目下有以下功能
   2.1、清空画板
   2.2、开始画图
   2.3、获取使用帮助
   2.4、保存文件
   2.5、退出程序
   按照您的需要输入项目前的数字即可。
